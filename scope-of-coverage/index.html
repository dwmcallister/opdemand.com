OpDemand Scope of Coverage

If you’ve purchased support from OpDemand, we…

Support

  • Configuration of hosting providers and nodes for the platform
  • Installation of platform controller and command-line tools
  • Deployment of nodes used to host platform services
  • Deployment of Twelve Factor applications on the platform
  • Scaling and management of application containers
  • Connecting applications to backing services like databases, caches and queues
  • Configuration of platform routing and proxies
  • Aggregation of log data for the platform and hosted applications
  • Standard administrative tasks within the platform

Do Not Support

  • Problems with backing services like databases, caches and queues
  • Issues related to cloud providers or underlying infrastructure
  • Integration with unsupported providers or features
  • Problems with Docker or Chef not related to Deis
  • Customization of core platform source code
