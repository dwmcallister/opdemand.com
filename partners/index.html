Technologies

Deis is powered by best-of-breed DevOps tools including Docker, CoreOS and Heroku Buildpacks.

Deis uses Docker’s revolutionary Linux container engine. Docker allows Deis to build, distribute and execute industry-standard Linux containers.

Deis is powered by CoreOS, the operating system of the future. CoreOS’s minimal footprint, secure auto-update technology and first-class Docker support help Deis deliver a 'private Heroku' that is built to scale.

Deis is deeply inspired by Heroku and Twelve Factor methodology. Deis developers participate in the Heroku community and contribute to Heroku's open source Buildpacks.

Hosting Partners

Deis can be deployed anywhere, including every public cloud, private cloud or bare metal. OpDemand works closely with hosting partners to ensure Deis runs on their infrastructure.
